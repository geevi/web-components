# Encapsulate the web ## Ryan Seddon # ME * [@ryanseddon](https://twitter.com/ryanseddon) * [thecssninja.com](http://www.thecssninja.com/) * [Modernizr](http://modernizr.com/) * [SEEK](http://seek.com.au/) # Web Components, what are they * Allow authors to compose re-usable "widgets" # Using a UI widget today 1. Get widget source 2. Include in page. May require CSS, JavaScript and Some initialisation code 3. Check to make sure it hasn't broken anything 4. Fix broken things # The future Web Component way 1. Import component 2. Use on page ```html Contents ``` # Web Components is the umbrella term * Custom Elements * Shadow DOM * Templates * Imports * Decorators # What problems do they solve * Encapsulation - like iframes without the baggage * Modularisation * Easy solution powerful mechanism for other developers to use, see source of slides. # Examples * This slide deck is a series of components * <x-mardown /> * <x-slideshow /> * <x-slide /> which extends <x-mardown /> * <syntax-highlight /> # Component lifecycle * Registering on page load `document.register()` * registering on createElement `document.createElement('x-markdown')` * as constructor e.g. `new XMarkdown()` # Custom Elements * Extending existing elements * creating new ones # Shadow DOM * Where the encapsulation happens * Browser vendors have had this functionality for a long time * Gives markup and style encapsulation safe from 3rd parties # Templates * Inert chunk of DOM * Allows you to nicely create a template that the component will use * Won't become active until it's injected into an element # Imports * How a component is included in the document # Decorators * Enhance or override the the presentation of an exisiting element * Applied via CSS decorator: url(#idname) * Unlike a custom element with a shadow DOM applied the shadow DOM in a decorator is never accessible via script. # Polymer ## Handy polyfills + extra layer on top * Wraps up web components into nice library * Handling events * Model Driven Views (MDV) think Angular.js but native * Much more # Thanks * Slides: [thecssninja.com/talks/wdcnz13](http://www.thecssninja.com/talks/wdcnz13) * Polymer: [polymer-project.org](http://polymer-project.org/)
